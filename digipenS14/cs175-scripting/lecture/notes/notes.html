Classes 1

What is a class?

a class is an abstract representation of an object. It stores information about the types of data that an object can hold and the behavious that an object can exhibit. Essentially, when you create a class you are creating a custom type that contains other types and functions. Clase allow you to better control how objects are created and how they interact with one another.

What is a Class Constructor?

A constructor is a function inside a class what has the same name as the class. It will be called only once, when you create an object of type that class. The constructor can take parameters but can never have a return type (since by default it should return an object of that class’ type).

Lab Exercise

Create a hero class using the following steps:

  1. Game.fla
  2. Hero Class .as file, inside an a folder GameObjects
    • PosX:Number
    • PosY:Number
    • iHealth:int
    • sName:String
    • Constructor(____, ____, ____)
  3. Test Hero Class by tracing out the xPos, yPos and iHealth
  4. Create an Enemy class that has everything except the iHealth

Hero.as
    
    package GameObjects
    {
        public class Hero
        {
            //PROPERTIES
            public var nPosX:Number;
            public var nPosY:Number;
            public var iHealth:int;
            public var sName:String;

            //METHODS
            public function Hero(_nPosX:Number, _nPosY:Number, _iHealth:int):void
            {
                nPosX = _nPosX;
                nPosY = _nPosY;
                iHealth = _iHealth;
                sName = "Hero";
            }
        }
    }
Game.fla
    
    //IMPORT CLASS
    import GameObjects.Hero;

    //CREATE CLASS OBJECTS
    var Player:Hero = new Hero(200,200,100);

    //TRACE
    trace(Player.sName);
    trace("Player is located at x = " + Player.nPosX);
    trace("Player is located at y = " + Player.nPosY);
    trace("Player has health = " + Player.iHealth);

Game State Manager—Cycle Functions

The Game State Manager should always be in a state. This state can be any one of the 5 cycle functions:

  • Create
  • Initialize
  • Update
  • Uninitialize
  • Destroy
