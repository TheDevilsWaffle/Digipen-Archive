    Super Space Race
    a UX/UI project in the guise of a space flight game

    BACKGROUND

    Super Space Race is a UX/UI prototype project for me to test my skills at building and designing menus, user interfaces, and create an experience that feels like a space flight simulator. While there is no game yet, the goal of the project is to nail the sensation of feeling like you are flying through space quickly in a hi-tech space ship.
    Super Space Race Raw Gameplay

    DETAILS

    Super Space Race is currently being made in Unity 3D and scripted in C# with several specific goals:

    * Create a multi-input (keyboard, mouse, XBox controller) capable menu system
    * Create a hi-tech looking, functional space ship HUD
    * Create a ship controller that gives the sensation of flying a ship fast through space
    * Script a flexible input system that works with a mouse, keyboard, and XBox Controller
    * Utilize a custom event system to communicate in game events to the HUD
    * Maximize the visual potential of the game using camera effects, custom shaders, and particles systems
