Looping

Looping Statements allow you to perform a specific block of code repeatedly using a series of values or variables. Always enclose the block of code in braces “{}”.

While


	while (expression)
	{
		statements
	}
	following statements

If ‘expression’ is true, the statements are executed until it becomesfalse. However, If ‘expression’ is false, the execution resumes at the following statement. The expression is evaulated before the statement is executed. When the expression is false from the first time, the statement is never executed.

Do…While


	do
	{
		statements
	}
	while (expression);

The statements are executed at least once before the expression is checked. If the expression is true, the statement is executed and the expression is evaluated. This will repeat until the statement becomes false.

For


	for(expression1; expression2; expression3)
	{
	statements;
	}
	following statements

  • Expression 1: Initializes the loop.
  • Expression 2: The test made before each iteration.
  • Expression 3: Evaluated after each iteration.
The comma operator can be used to put multiple expressions within parts of the for statement. Example: 

	for(var i:int = 0, va k:int = 10; i < 5; i++, k += 3)
	{
		trace(i + k);
	}

Object-Oriented Programming

Introduction

Object-Oriented Programming, or OOP, is a way of organizing the code in a program by grouping it into objects. Objects are individual elements that include information (data values and functionality)

Attributes

How can we build an object for a character using OOP? We do this by defining its properties. For example:

  • Shape
  • Position X Y
  • Scale X Y
  • Rotation
  • Health
  • Speed
  • etc…

Now what can the character do, what are its functionalities or actions? These are called methodsFor Example:

  • Move
  • Jump
  • Shoot
  • Play Animation
  • etc…
Attributes are both properties and methods.

Constructor

The constructor is a function that has the same name as the object type and it used to create the object.

Dot Operator (.)

The dot operator is used to access the attributes inside the object.

Strings

In programming a string is a text value. The value can be a sequence of letters, numbers, or other characters strung together into a single value. For instance, the line of code below creates a variable with the data type String and assigns a literal string value to that variable:


	var sAlbumName:String = "3 for the money";

Creating a string

There are multiple ways to create a string:


	//creating a string
	var str1:String = new String("hello");//using double quotes

	var str1:String = new String('hello');// using single quotes

Now using the constructor:


	// creating a string using the constructor
	var str1:String = new String("hello"); //assigns "hello" to str1

	var str2:String = new String(str1); //assigns the value of str1 to str2

	var str3:String = new String(); //assigns a blank string to str3

Escape Sequence Characters

  • \n: Newline
  • \r: Carriage Return

  • \unnnn: The unicode character with the character code specified by the hexadecimal number nnnn; for example, \u263a is the smiley face character
  • \': single quotation mark
  • \": double quotation mark
  • \\: single backslash character

Attributes

As with every object, String has attributes (properties & methods).

Properties

The property that the String object has is length:int which is equal to the number of characters in the string (an empty string has a length of 0).


	var myStr:String = "Hello";
	trace(myStr.length); // will return 5

Methods

There are many methods inside the String object. Using methods will make your life way easier. Important things to remember when using a method are:

  • Read its description in the help
  • Know what kind of arguments you need to send the function
  • Know what the function returns to you
  • Know if the function changes the original object or returns a new object (sometimes both)
  • Use the API examples in the documentation to understand the how the method works
charAt()

This method examines the individual characters in various positions in a string and returns the character located at the user defined index.

Example:

//using charAt()
var myString:String = "Waffles are yummy!
trace(myString.charAt(6)); // traces out ‘s’ as it is at index[6] in myString
Parameters:

1—integer, index location of the string

Return:

a single string character

charCodeAt()

This method examines the individual characters in various positions in a string and returns the character as an Unicode character value.

Example:

	//using charAt()
	var myString:String = "Waffles are yummy!";
	trace(myString.charAt(6)); // traces out "s" as it is at index[6] in myString
Parameters:

1—integer, index location of the string

Return:

a single string character

concat()

Acts like string addition by taking two strings and joining them sequentially into one.

Example:

	//using concat()
	var myString1:String = "Waffles";
	var myString2:String = "are";
	var myString3:String = "yummy!";
	var myString4:String = myString1.concat(" ", myString2, " ", myString3);
	trace(myString4); // traces out "Waffles are yummy!"
Parameters:

At least 1variable containing a string, but multiple variables with strings can be used separated by commas. Newly formed strings can also be used as parameters.

Return:

String—A new string consisting of this string concatenated with the parameters used.

"+" & "+="

A different form of concat(), which also acts like string addition by taking two strings and joining them sequentially into one.

Example:

	//using ‘+’ with strings
	var myString1:String = "Waffles";
	var myString2:String = "are";
	var myString3:String = "yummy!";
	var myString4:String = myString1 + " " + myString2 + " " + myString3;
	trace(myString4); // traces out "Waffles are yummy!"

	//using ‘+=’ with strings
	var myString1:String = "Waff";
	myString1 += "les";
	trace(myString1); // traces out "Waffles"
Parameters:

At least 1variable containing a string, but multiple variables with strings can be used separated by commas. Newly formed strings can also be used as parameters.

Return:

String—A new string consisting of this string concatenated with the parameters used.

substr()

Takes two parameters: the first parameter defines what position to start from in a string, the second parameter defines how many positions after that to take.

substring()

Takes two parameters: the first parameter defines what position to start from in a string, the second parameter defines the second position to end at from the string.

slice()

Similar to the substring()method, except it can take negative integers as parameters to traverse backwards in a string. It takes two parameters: the first parameter defines the first position in a string, and the second parameter defines the second position in a string.

indexOf()

Used to locate matching substrings within a string. Needs one parameter which is the words or letters of the string you are trying to find. The words or letters you choose to find are case-specific and upon finding them, the method will return the index location of where those words or letters start. It can take a second parameter to define what position in the string to start from.

lastIndexOf()

Exactly the same as the indexOf() method, but it finds the last occurence of the words or letters that you define to find in a string.

split()

Used to create an array of substrings, which is divided based on a defined delimiter. The first parameter it takes is the defined delimiter. A second parameter may also be defined, which defines the maximum size of the array that is returned.

Truncating Elements

You can truncate an Array using an array's length() property. If you set the length property of an array to a length that is less than the current length of the array, the array will be truncated and remove any elements stored at index numbers higher than the new value of the length minus 1.
